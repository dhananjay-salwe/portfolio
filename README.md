# Dhananjay Salwe - Portfolio Website

A modern, responsive portfolio website showcasing my work as a Full Stack Developer. Built with React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 Features

- **Modern Design**: Clean, professional design with dark/light theme support
- **Responsive**: Mobile-first design that works on all devices
- **Smooth Animations**: Powered by Framer Motion for engaging user experience
- **Featured Projects**: Showcase of DataAuth, Attendo, and AI/ML projects
- **Skills Section**: Categorized display of technical skills
- **Experience Timeline**: Professional journey with achievements
- **Contact Form**: Easy ways to get in touch

## 🛠️ Built With

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS v3
- **Animations**: Framer Motion
- **Routing**: React Router DOM
- **Icons**: Custom SVG icons
- **Deployment**: Ready for Vercel, Netlify, or any static hosting

## 📱 Sections

1. **Hero** - Introduction with call-to-action buttons
2. **Projects** - Featured projects with live demos and GitHub links
3. **Skills** - Technical skills categorized by domain
4. **Experience** - Professional timeline with achievements
5. **About** - Personal story and interests
6. **Contact** - Multiple ways to connect

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/dhananjay-salwe/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
